This portfolio of GIS applications showcases some of my work—projects in which I've taken the Lead Developer role.

Of course, good looks aren't everything. My UIs are complemented by efficient server-side and client-side code (I use a variety of tools like ColdFusion, PHP, Python and Node.js), in order to handle the complex GIS data that is backed by ArcGIS Enterprise and Microsoft SQL resources.
